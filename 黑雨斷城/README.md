# 《黑雨斷城》作品目錄

## 作品定位

- 類型：寫實末日求生、都市斷鏈、團隊生存、軍事常識
- 核心賣點：沒有系統、沒有喪屍、沒有無限物資。每一次錯誤判斷，都會拿水、體溫、時間甚至人命付帳。
- 目標受眾：喜歡硬核求生、災難推演、團隊決策與有聲小說的 YouTube 觀眾
- 單章長度：三千至四千五百個中文字，約十二至十八分鐘朗讀

## 目錄規則

本作品所有資料都收在 `黑雨斷城` 目錄內，避免與其他小說混用。

```text
黑雨斷城/
  README.md
  novel-bible/
    story-bible.md
    characters.md
    survival-knowledge.md
    plot-outline.md
    retention-design.md
    timeline.md
  chapters/
    ch001.md
    ch002.md
  image-prompts/
  youtube/
  assets/
    thumbnails/
    generated-images/
    audio/
    video/
```

`chapters/` 內一章一檔，每個 `chXXX.md` 都是可直接用於 TTS 或錄音的正式有聲書稿。不另外建立閱讀版、完整合併稿或朗讀稿複本。

## 內容原則

1. 求生知識必須由劇情觸發，不寫成課本。
2. 每章只讓觀眾牢記一至三個核心知識點。
3. 主角可以判斷正確，但不能永遠零代價。
4. 軍事內容聚焦紀律、通訊、警戒、地形、後勤與救護，不提供武器製作或傷害他人的操作教學。
5. 涉及急救、飲水與災害應變時，正式發布前應依台灣消防署、衛福部及紅十字會等可靠來源再次查證。
