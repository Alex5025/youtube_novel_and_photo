# 《末日重生：我的庇護所無限升級》作品目錄

## 作品定位

- 類型：末日重生、系統升級、庇護所經營、復仇逆襲
- 核心賣點：主角回到災變前三小時，利用前世情報與升級系統建立庇護所，改寫被背叛致死的結局。
- 內容用途：繁體中文 YouTube 連載有聲書

## 目錄規則

```text
末日重生_我的庇護所無限升級/
  README.md
  novel-bible/
  chapters/
    ch001.md
    ch002.md
    ch003.md
    ch004.md
    ch005.md
    ch006.md
  image-prompts/
  youtube/
  assets/
    thumbnails/
    generated-images/
    audio/
    video/
```

`chapters/` 內一章一檔，每個 `chXXX.md` 都是可直接用於 TTS 或錄音的正式有聲書稿。不建立閱讀版、完整合併稿或內容重複的朗讀稿。

生圖提示詞與 YouTube 包裝使用相同章號，例如 `image-prompts/ch001-prompts.md` 與 `youtube/ch001-youtube.md`。
